# 🔄 AUTHENTICATION FLOW

```bash
User Input Credentials
        ↓
POST /login
        ↓
Server Validation
        ↓
JWT Token Generated
        ↓
Stored in Browser (LocalStorage)
        ↓
Access Granted → /dashboard
```
